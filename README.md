# 🌱 Spring Framework Essentials

> This repository contains the code examples, solutions, and exercises for
> the [Spring Framework Essentials](https://spring.academy/courses/spring-framework-essentials) course on Spring
> Academy.

## 📖 About the Course

Spring Framework Essentials is a hands-on course designed to teach the core concepts of the Spring Framework through a
series of guided labs and practical exercises.

Enrollment is free, and all lab instructions can be found directly within the course.

## 📂 Repository Structure

Each directory in this repository corresponds to a specific **Lab** from the course.

The labs are organized sequentially and are intended to be completed in the order presented in the course material. Each
lab focuses on a particular Spring Framework concept and builds upon knowledge gained in previous exercises.

### 🧪 Labs & Progress

Every finished lab is paired with a dedicated pull request. These pull requests contain:

- The lab objectives
- Required tasks and instructions
- TODOs that are completed during the lab exercises

This setup makes it easy to track progress and understand the expected outcome for each lab.

## 🛠️ Building the Project

This repository provides build configurations for both:

- **Gradle** (`build.gradle`)
- **Maven** (`pom.xml`)

You can use whichever build tool you prefer to compile, run, and test the project.

### Using Gradle

```bash
./gradlew build
```

### Using Maven

```bash
./mvnw verify
```

## 🚀 Getting Started

1. Clone this repository.
2. Enroll in the Spring Framework Essentials course.
3. Open the lab corresponding to the course section you are currently working on.
4. Follow the lab instructions from the course.
5. Complete the TODOs and verify your solution using your preferred build tool.
